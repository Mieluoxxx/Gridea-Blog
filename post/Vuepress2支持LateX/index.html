Vuepress2支持LateX.md

· 2023-10-29 · # Blog

在你的项目中安装相关的Tex包 katex 或 mathjax-full:

yarn add -D katex
# or
yarn add -D mathjax-full

之后启动它

// .vuepress/config.ts
import { mdEnhancePlugin } from "vuepress-plugin-md-enhance";

export default {
  plugins: [
    mdEnhancePlugin({
      // 使用 KaTeX 启用 TeX 支持
      katex: true,
      // 使用 mathjax 启用 TeX 支持
      mathjax: true,
    }),
  ],
};

::: warning
你只能启用其中一个，并且 katex 具有更高的优先级。
:::
